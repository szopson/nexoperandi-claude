```markdown

## TL;DR
[3 bullet points z głównymi wnioskami]

## [Sekcja 1 - problem]
[Opisz problem który rozwiązujesz]

## [Sekcja 2 - rozwiązanie]
[Przedstaw rozwiązanie]
```

### On-Page SEO Checklist
- [ ] Keyword in title (H1)
- [ ] Keyword in first paragraph
- [ ] Keyword in at least one H2
- [ ] Internal links (2-3)
- [ ] External links (1-2 authoritative sources)
- [ ] Meta description with CTA
- [ ] Alt text on images
- [ ] URL contains keyword

## Quality Standards

- **Blog posts:** 800-1500 words
- **LinkedIn posts:** 150-300 words
- **X posts:** 280 characters max (threads: 5-10 tweets)
- **Newsletter:** 500-800 words

Every piece must have:
1. Clear value proposition
2. Specific examples or data
3. Single CTA
4. Proofread for errors
